# 負の二項分布で読み解くブランド別購買行動：平均だけでは見えない市場構造

## はじめに

マーケティングにおいて、ある商品カテゴリの売れ行きを把握する最も素朴な方法は「平均購入回数」を見ることである。しかし、平均だけでは市場の全体像は見えない。たとえば、「年間平均3回購入されるブランド」が2つあったとして、一方は多くの消費者が少しずつ買い、他方は少数の固定客が繰り返し買っているかもしれない。

本記事では、森岡毅・今西聖貴『確率思考の戦略論』で論じられているフレームワークを参考に、消費財の購買データに**負の二項分布（NBD: Negative Binomial Distribution）**をフィッティングし、$M, K$ パラメータの解釈を通じてブランドの購買構造を分解する手法を紹介する。

## データ

Dunnhumby が公開している "The Complete Journey" データセット（Kaggle）を使用した。約2,500世帯の2年間にわたるスーパーマーケット購買履歴を含む。本分析では以下の条件でデータを絞り込んだ。

- **カテゴリ**: SOFT DRINKS
- **期間**: 最初の52週間（DAY 1〜364）に限定し、定常性をある程度確保
- **ブランド**: MANUFACTURER コードの上位5メーカー + Others の6グループに集約

フィルタリング後の基本統計は以下の通り。

| 項目                         | 値      |
| ---------------------------- | ------- |
| SOFT DRINKS 商品数           | 1,704   |
| 52週間のトランザクション数   | 56,090  |
| 世帯数                       | 2,272   |
| 平均購入回数（カテゴリ全体） | 15.11回 |
| 分散                         | 406.38  |

## 手法：負の二項分布モデル

### モデルの構造

NBD モデルは、消費者の購買行動に関する2つの仮定から導出される。

**仮定1（個人レベル）**: 各消費者 $i$ の購入はポアソン過程に従い、期間 $T$ における購入回数は購入率 $\lambda_i$ のポアソン分布に従う。

$$
x_i \mid \lambda_i \sim \mathrm{Poisson}(\lambda_i T)
$$

**仮定2（集団レベル）**: 消費者間の購入率の異質性をガンマ分布で表現する。

$$
\lambda_i \sim \mathrm{Gamma}(K, \alpha), \quad f(\lambda; K, \alpha) = \frac{\alpha^K}{\Gamma(K)} \lambda^{K-1} e^{-\alpha\lambda}
$$

ここで $K$ は形状パラメータ（消費者の異質性を表す）、$\alpha$ はレートパラメータである。

$\lambda$ を積分消去すると、観測される購入回数の分布として負の二項分布が得られる。ここで $M = T \cdot K/\alpha$（プリファレンス：期間 $T$ における集団全体の平均購入回数）と置くと、PMF は次の **M, K 表記**で書ける。

$$
P(X = x) = \binom{x + K - 1}{x} \left(\frac{K}{K+M}\right)^K \left(\frac{M}{K+M}\right)^x
$$

### パラメータの意味

- **$M$（プリファレンス）**: 集団全体の平均購入回数。ブランドへの選好の強さを直接表す。
- **$K$（形状パラメータ）**: 小さいほど消費者間の購買率のばらつきが大きい。$K$ が小さいと「大量に買う人」と「ほとんど買わない人」の差が顕著になる。$K \to \infty$ でポアソン分布（消費者が均一）に収束する。
- **平均**: $\mu = M$
- **分散**: $\sigma^2 = M + M^2/K$。$K$ が小さいほど過分散が強まり、ポアソン分布（$\sigma^2 = M$）から乖離する。

### なぜポアソンではなく NBD なのか

ポアソン分布は平均と分散が等しい（$\sigma^2 = \mu$）ことを前提とする。しかし実際の購買データでは、分散は平均をはるかに上回る（過分散）。本データでは分散/平均比が **26.9** であり、ポアソンの仮定は全く成り立たない。NBD はガンマ分布による消費者間の異質性を明示的にモデリングすることで、この過分散を自然に捉えることができる。

## 結果

### カテゴリ全体のフィッティング

SOFT DRINKS カテゴリ全体（2,272世帯）に NBD をフィッティングした結果は以下の通りである。

| パラメータ                   | 推定値 |
| ---------------------------- | ------ |
| $M$（プリファレンス）        | 15.10  |
| $K$（形状パラメータ）        | 0.9905 |
| 平均（モデル）               | 15.10  |
| 平均（データ）               | 15.11  |
| $\lambda$ の変動係数 $= 1/\sqrt{K}$ | 1.00   |

$K \approx 1$ はガンマ分布が指数分布に近い形状であることを意味し、少数のヘビーバイヤーと多数のライトバイヤーが混在する典型的な市場構造を示している。

### ブランド別のフィッティング

各ブランドの購入回数データ（0回を含む）に個別に NBD をフィッティングし、以下の指標を算出した。

- **浸透率（Penetration）**: 全世帯のうち、そのブランドを1回以上購入した世帯の割合 $= 1 - P(X=0)$
- **購入頻度（Buying Rate）**: 購入者に限った平均購入回数 $= \mu / \text{Penetration}$

| ブランド | $M$（平均） | $K$（形状） | 浸透率（モデル） | 浸透率（実測） | 購入頻度 |
| -------- | ----------- | ----------- | ---------------- | -------------- | -------- |
| Mfr_1208 | 5.01        | 0.577       | 73.0%            | 76.6%          | 6.86     |
| Mfr_103  | 5.90        | 0.489       | 71.6%            | 75.5%          | 8.25     |
| Mfr_2224 | 3.13        | 0.542       | 64.5%            | 67.7%          | 4.84     |
| Mfr_69   | 3.33        | 0.213       | 45.0%            | 46.4%          | 7.40     |
| Others   | 1.10        | 0.302       | 37.2%            | 38.3%          | 2.97     |
| Mfr_2    | 0.26        | 0.157       | 14.4%            | 14.5%          | 1.84     |

モデルによる浸透率と実測値はよく一致しており、NBD が各ブランドの購買パターンを良好に捕捉していることがわかる。

## 考察

### 平均購入回数が近いブランドの比較

最も興味深い知見は、**Mfr_2224 と Mfr_69 の比較**にある。両者の平均購入回数はほぼ同じ（3.13 vs 3.33）だが、内部構造は大きく異なる。

| 指標                  | Mfr_2224 | Mfr_69 |
| --------------------- | -------- | ------ |
| $M$（平均購入回数）   | 3.13     | 3.33   |
| $K$（形状パラメータ） | 0.542    | 0.213  |
| 浸透率                | 64.5%    | 45.0%  |
| 購入頻度              | 4.84     | 7.40   |

**Mfr_2224** は比較的多くの世帯が購入しており（高浸透率）、1人あたりの購入頻度は控えめである。一方、**Mfr_69**（プライベートブランドと推定される）は浸透率が低いが、購入者は高頻度で繰り返し買っている。$K = 0.213$ という小さな値は、消費者間の購買率の異質性が極めて大きいことを意味し、「買う人は大量に買うが、買わない人は全く買わない」というパターンを示している。

このように、平均購入回数という単一の指標では同程度に見えるブランドでも、NBD のパラメータ分解によって**ライトユーザーも多く幅広いユーザーが一定回数購入している状況**と**一部のヘビーユーザーが売上を牽引している状況**を明確に区別できる。

### マーケティング施策への示唆

- **ヘビーユーザーに偏ったブランド**（Mfr_69）: 既存顧客のロイヤルティは高いが、新規顧客の獲得に課題がある。トライアル促進策（サンプリング、初回割引など）が有効と考えられる。
- **浸透率が高いがヘビーユーザーが少ないブランド**（Mfr_1208, Mfr_103）: 広く認知されており、頻度向上のためのリピート施策（ポイント還元、定期購入など）が考えられる。

### モデルの限界

本分析にはいくつかの限界がある。

1. **カテゴリ全体のフィッティング精度**: カテゴリ全体では、モデルの分散（245.44）がデータの分散（406.38）を下回った。これは NBD の2パラメータでは捉えきれない異質性（セグメント構造など）の存在を示唆している。
2. **定常性の仮定**: 52週間を単一の定常期間として扱ったが、プロモーションや季節性の影響は考慮していない。

## まとめ

本記事では、負の二項分布モデルを用いてSOFT DRINKSカテゴリの購買データを分析した。得られた知見は以下の通りである。

1. **ポアソン分布では過分散を捉えられない**。実データの分散/平均比は26.9であり、消費者間の購買率の異質性を明示的にモデリングする必要がある。
2. **形状パラメータ $K$ の解釈によりブランドごとの売上の立ち方の違いを解釈できる**。$K$ が小さいブランドは購入回数の分布が右裾に重く、少数のヘビーバイヤーが売上の大部分を占める構造である。逆に $K$ が大きいほど分布はポアソンに近づき、購買率のばらつきが小さく「まんべんなく買われるブランド」であることを意味する。平均 $M$ だけでは区別できないこの構造的な違いを、NBD は $K$ という単一のパラメータで定量化する。
