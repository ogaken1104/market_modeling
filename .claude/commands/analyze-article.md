# Phase 3: 記事の執筆

## 入力

記事の骨子を読み込む：

@dev/prompt.md

---

分析の仕様・完了ステップを確認する：

@dev/spec.md

## タスク

上記の情報と `src/analysis.ipynb` の分析結果をもとに、`article/output.md` を執筆する。

### 記事の構成（prompt.md の記事骨子に従う）

1. 導入：なぜ統計モデルで市場をモデリングするのか
2. データ：使用データセットの説明・絞り込み条件
3. 手法：各モデルの数式・パラメータの意味を解説
4. 結果：フィッティング結果・可視化の説明
5. 考察：マーケティング施策への示唆
6. まとめ
7. 参考文献

### 執筆ルール

- **文体は常体**（〜だ、〜である）
- 数式は LaTeX 形式（`$$...$$` でブロック、`$...$` でインライン）
- `src/analysis.ipynb` で得られた具体的な数値・グラフの読み取り結果を引用する
- 数学に慣れている読者を前提とし、式を省略しない
- マーケティング施策への示唆は分析結果から論理的に導く

### 分析ノートブックの確認

- `src/analysis.ipynb` を読み込み、各セルの出力（数値・表・グラフの解釈）を記事に反映する
- 完了した spec.md のステップと対応させながら結果セクションを構成する

## 手順

1. `dev/prompt.md` の記事骨子・ゴールを確認する
2. `src/analysis.ipynb` の実行結果を読み取り、数値・解釈を整理する
3. `article/output.md` のドラフトを作成してユーザーに提示する
4. フィードバックを反映して最終版を保存する
